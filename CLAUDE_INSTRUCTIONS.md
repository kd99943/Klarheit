Tech Stack: Java 17/21, Spring Boot 3.x, Spring Web, Spring Data JPA, MySQL Driver, Lombok.

Initialize Project: Generate the base Spring Boot project structure.

Database Configuration: Configure application.yml for a MySQL connection. Create a foundational SQL initialization script (schema.sql or Liquibase/Flyway setup) for the following core entities:

Product (id, name, material, base_price, image_url)

LensOption (id, type, index_value, additional_price)

Prescription (id, user_email, sph_od, sph_os, cyl_od, cyl_os, axis_od, axis_os, pd)

Order (id, order_number, total_amount, status)

Build the controllers and services, adhering strictly to RESTful principles.

Catalog API: Create ProductController with a GET /api/v1/products endpoint to serve the 4 flagship glasses to the Collections page.

Order & Config API: Create OrderController with a POST /api/v1/orders/checkout endpoint. This must accept a complex JSON payload containing the selected frame, lens options, and the intricate prescription data from the Config Lab, calculate the final total securely on the backend, and persist it to MySQL.

Frontend Integration: In the React frontend, create a /src/services/api.ts file using fetch or axios to connect the UI components to these new Spring Boot endpoints. Replace all static mock data with dynamic API calls.
