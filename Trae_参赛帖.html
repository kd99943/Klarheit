🔬 算法与物理级渲染交互沙盒 (Technical Sandbox)

算法实现说明：在 WebAR 试戴中，我们使用左眼(33)、右眼(263)、鼻梁(168)和鼻尖(1)的 3D 空间坐标构建正交基底矢量，解算出精确的旋转四元数 (THREE.Quaternion)。这让眼镜在侧脸转动达 90° 时依然能完美同步贴合。

// 3D姿态正交基计算
const dirX = R.clone().sub(L).normalize();
const dirY_temp = B.clone().sub(N).normalize();
const dirZ = dirX.clone().cross(dirY_temp).normalize();
const dirY = dirZ.clone().cross(dirX).normalize();
rotMatrix.makeBasis(dirX, dirY, dirZ);
quaternion.setFromRotationMatrix(rotMatrix);

材质物理属性 (Index of Refraction)：我们使用了 Three.js 的 MeshPhysicalMaterial。通过调节镜片的折射率 (IOR) 和厚度，可以细腻呈现光线穿过玻璃边缘的物理弯曲和倒角反光。

物理折射率 (IOR) 1.56
new THREE.MeshPhysicalMaterial({
  transmission: 0.95,
  ior: 1.56, // 折射率
  thickness: 0.04, // 镜片物理厚度
  roughness: 0.0,
  metalness: 0.0
});

减反射膜 (Anti-Reflective Coating) 仿真：高端眼镜通常带有防反射反射蓝膜。我们在镜片上覆盖了一层极薄的 Sheen（光泽）膜，并使用蓝绿色或紫色调，完美模拟高级镀膜质感。

减反射蓝膜反射强度 (Sheen Intensity) 1.0
镀膜反射颜色：
new THREE.MeshPhysicalMaterial({
  sheen: 1.0,
  sheenColor: new THREE.Color("#06b6d4"),
  sheenRoughness: 0.15,
  reflectivity: 0.8
});

柔光环境与 PCF 软阴影：我们使用自定义摄影棚灯光，并开启 PCF 软阴影 (radius: 4)，让镜框在脸部或鼻梁上投下自然的漫反射软阴影，避免生硬黑边。拖动滑块可以旋转光源方向！
